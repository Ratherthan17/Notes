用Docusaurus和Github搭建网站

我搭建时的环境 

win10
Node.js v18.18.0
npm 9.8.1
yarn 1.22.19

2023年10月7日

一、下载Node.js

· Node.js官网

:::caution注意

  • Node.js得是16.14及以上的版本，安装好后，可以在命令行输入node -v敲回车查看安的版本。
  • Node.js只支持Windows8.1及以上，Windows7不能用。原因是node.js的版本太高了，v12.16.2以上版本不支持win7系统，可以下载12.16.2及以下的版本[1]。
    :::

安装yarn

npm install -global yarn
yarn config set registry https://registry.npm.taobao.org/ # 使用淘宝维护的npm镜像源[2]

二、创建本地站点

1. 打开命令行窗口

找个地方新建一个文件夹，点进去（这些文件、文件夹的名字最好用英文名，不然可能会出问题）。
点一下地址栏，输入cmd，然后回车——在当前目录下打开命令行窗口


2. 输入命令创建站点 

npx create-docusaurus@latest my-website classic

:::tip提示
这个"my-website"可以自由命名，"classic"是选用经典模板进行创建
:::

下面是创建成功的图片

最下面有提示，输入

cd my-website
npm start


它会有一个绿色的条，等条走完了，会自动打开一个 http://localhost:3000 的网站，这个网站和官方文档 5-minute tutorial 网站一模一样，因为创建时用的是经典模板



:::caution注意
要想在浏览器里看创建的网站，就必须得让之前输入npm start的控制台窗口运行着，要是把它关了，网站就打不开了。
:::

1. 在GitHub上新建库

  • 在GitHub上新建一个名为 你的账户名.github.io 的仓库
  • 进入仓库，点击settings，

当初遇到的错误

最开始我用官方文档"Fast Track"中的这个命令创建站点老出错 

npx create-docusaurus@latest my-website classic

最后偶然发现他们5-minute tutorial教程里的命令能用 

npm init docusaurus@latest my-website classic

现在这个网站就是用上面的命令创建出来的那个。之后我在写这个笔记的时候，为了截图片，又用"Fast Track"中的命令创了一次，结果成功了，真是莫名其妙！
